# The-Weather-App

🌦️ Advanced Weather App

Um aplicativo de previsão do tempo moderno, com foco em experiência do usuário, performance e design, inspirado em produtos como Apple Weather e The Weather Channel.

Este projeto foi desenvolvido com JavaScript puro, explorando conceitos fundamentais de frontend enquanto implementa funcionalidades reais de aplicações de grande escala.

# ✨ Funcionalidades
🔍 Busca inteligente com autocomplete
Sugestões de cidades em tempo real conforme o usuário digita
📍 Geolocalização automática (GPS)
Detecta a localização do usuário ao abrir o app
🌡️ Dados climáticos em tempo real
Temperatura atual
Umidade
Velocidade do vento
Cobertura de nuvens
Probabilidade de chuva
🌗 Tema dinâmico (dia/noite + clima)
Interface muda automaticamente com base nas condições reais
🎞️ Animações visuais com Canvas
Chuva animada
Partículas solares
Fundo dinâmico responsivo
⭐ Sistema de favoritos
Salva cidades no navegador (localStorage)
Acesso rápido com um clique
⚡ Debounce na busca
Otimização de performance para evitar requisições desnecessárias
🎨 UI/UX
Design inspirado em apps modernos (Apple / iOS)
Glassmorphism (blur + transparência)
Micro-interações (hover, animações suaves)
Layout responsivo e limpo
Hierarquia visual bem definida
# 🧠 Tecnologias utilizadas
HTML5
CSS3 (Glassmorphism + animações)
JavaScript (Vanilla JS)
Canvas API (animações)
Open-Meteo API (dados climáticos)
Geocoding API (busca de cidades)
LocalStorage (persistência de dados)
⚙️ Conceitos aplicados
Manipulação de DOM
Consumo de APIs (fetch)
Estruturação de código (estado global)
Debounce e otimização
Renderização dinâmica de UI
Separação de responsabilidades (UI / lógica / dados)
Animações com Canvas
